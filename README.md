# Lista de Tarefas 

To-do list feita em React (com Typescript + Tailwind CSS)

### Instalação
- `npm install`

### Para ligar o servidor
- `npm run dev`

Com o servidor ligado clique em  [http://localhost:3000](http://localhost:3000) para acessar a lista de tarefas.
